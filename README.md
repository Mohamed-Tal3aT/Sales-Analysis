# Sales-Analysis#
Sales Analysis with NumPy

## الوصف
مشروع لتحليل بيانات المبيعات اليومية باستخدام **NumPy** فقط.  
يقوم البرنامج بـ:
- إنشاء بيانات المبيعات (مع بعض الأيام الشاذة).  
- إزالة الشواذ.  
- حساب المتوسط، الانحراف المعياري، والقيم القصوى والدنيا.  
- اكتشاف الأيام الشاذة مقارنة بالمتوسط والانحراف المعياري.  
- حساب المئينات وأيامها.  
- حساب المتوسط الأسبوعي لكل 7 أيام.

## المتطلبات
- Python 3.x  
- مكتبة NumPy
